## 🚀 Sobre el proyecto

**Ahexafy** es una plataforma de streaming musical inspirada en Spotify, desarrollada como proyecto personal.

🎯 Objetivo: crear una aplicación web moderna donde los usuarios puedan **escuchar, compartir y descubrir música** de forma intuitiva y atractiva.

💡 Este proyecto está en fase inicial (🚧 en desarrollo), pero se plantea como una aplicación completa y escalable.

---

## 🎯 Funcionalidades previstas

### 🎵 Música
- ▶️ Reproducción de canciones  
- ⏭️ Control de reproducción (play, pausa, skip)  
- 🔊 Control de volumen  

### ❤️ Interacción
- 👍 Dar like a canciones  
- ⭐ Guardar en favoritos  
- 📂 Crear playlists personalizadas  

### 👥 Social
- 👤 Seguir a otros usuarios  
- 📰 Feed de actividad  
- 💬 (Futuro) Comentarios o interacción social  

### 📤 Creadores
- ⬆️ Subir música  
- 🎤 Perfil de artista  
- 📊 Estadísticas básicas de reproducciones
